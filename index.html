Default Settings

{
	state: 'auto',
	slicesX: 6,
	slicesY: 5,
	slicesTime: 3000,
	waitingTime: 4000
}

and MUST add 

class="item"

to each img tag like this: 

<div class="slider-container">
	<img src="..." alt="image" class="item"/>
	<img src="..." alt="image" class="item"/>
	<img src="..." alt="image" class="item"/>
</div>

Simple Usage

 $('.puzzle-slider-1').puzzleSlider();

10 X 7

$('.puzzle-slider-2').puzzleSlider({
	slicesX: 10,
	slicesY: 7
});

'hover' state

 $('.puzzle-slider-3').puzzleSlider({
	state: 'hover'
});
